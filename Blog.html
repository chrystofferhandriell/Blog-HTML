      • O que é HTML e por que ele é indispensável na Web?

      Ele funciona como os “blocos de construção” de qualquer site. Sem HTML,
      nada poderia ser exibido no navegador. Aprender HTML significa entender como a web funciona desde sua base,
      permitindo que o aluno tenha domínio sobre como cada elemento é
      organizado dentro de uma página.

    • Como o HTML funciona na prática

      • O HTML utiliza tags — estruturas que indicam ao navegador como exibir cada parte da página.Alguns exemplos:

        • < h1> a < h6> — Títulos
        • < p> — Parágrafos
        • < img> — Imagens
        • < a> — Links
        • < div> e < section> — Estruturação de layout

      Essas tags funcionam como caixas que agrupam e identificam partes do conteúdo, permitindo ao navegador interpretar tudo corretamente.
